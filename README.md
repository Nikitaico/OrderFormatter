# OrderFormatter
Программа для структурирования хаотичного текста с номерами заказов.  Преобразует найденные последовательности цифр в формат ПРXXXXXX_VLH. Результат разбивается на строки до 255 символов. Кнопки перенесены наверх, добавлена кнопка копирования всего поля. Иконка приложения постоянно находится в трее Windows.
